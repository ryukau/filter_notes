# filter_notes
主に音の合成について調べたことを書いています。

[Index page](https://ryukau.github.io/filter_notes)

# ページのビルド
Python3 で書かれた `build.py` を実行すると html がビルドされます。 [Git](https://git-scm.com/) と [pandoc](https://pandoc.org/) が必要です。

```bash
python3 build.py
```
